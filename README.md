# مُدَبِّر الْأَسْرَارِ الْعُلْيَا

تطبيق قرآني متكامل بمعمارية Clean Architecture.

## المميزات

- القرآن الكريم مع البحث
- حاسبة الجُمَّل (حساب قيم الحروف العربية)
- الأسماء الحسنى الـ 99
- أسئلة البصيرة
- دعم RTL الكامل
- يعمل بدون إنترنت

## البناء

```bash
flutter pub get
flutter build apk --release
```
